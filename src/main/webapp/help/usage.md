## Browsing datasets
Use the Browse page to filter datasets by species, tissue, and condition, then open a dataset for detailed metadata.

## Searching genes
Search by gene symbol to see expression summaries and relevant datasets across the collection.

## Integrating datasets
Select multiple datasets and launch the integration view to compare cell populations in a shared embedding.

## Exporting results
Download filtered tables or DEG results directly from the Download page for offline analysis.

## Troubleshooting
If a view fails to load, refresh the page and verify that pop-ups are enabled for visualization windows.
